# CareerConnect

CareerConnect is a premium full-stack job portal experience for job seekers and recruiters.

This folder now contains only the frontend application. The backend is in the sibling `../backend` folder.

## Frontend

The frontend is built with React, Vite, Tailwind CSS, Framer Motion, Axios, React Router, and React Hot Toast.

### Run the frontend

```bash
npm install
npm run dev
```

The Vite app runs on `http://localhost:3000` by default.

## Backend

The backend lives in `../backend/`.

### Run the backend

```bash
cd backend
npm install
npm run dev
```

Set `MONGO_URI`, `JWT_SECRET`, and optionally `CLIENT_URL`.
